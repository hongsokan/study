# HTML


## HTML Style Guide

### 첫번째 줄에는 항상 문서 유형 선언
```html
<!DOCTYPE html>
```

### 요소 이름은 소문자로
```html
<h1></h1>
<p></p>
<div></div>
```

### 모든 HTML 요소는 닫아주기
```html
<section>
…
</section>
```

### 속성 이름도 소문자로
```html
<a href="https://www.w3schools.com/html/">HTML Link</a>
```

### 속성 값은 항상 따옴표로 감싸주기
```html
<table class="striped">
```

### 이미지는 src는 필수, width, height, alt 속성 값도 되도록 미리 추가
```html
<img src="image.png" width="100px" height="100px" alt="This is a sample image">
```
